# StUF-BG 2.04

StUF-BG 2.04 is afgeleid van de gegevensstandaard [GFO-Basisgegevens](./documenten/GFO_Basisgegevens.pdf). De
2.04 versie van het [BAG-GBA koppelvlak](https://vng-realisatie.github.io/StUF-BAG-GBA/) is op
deze versie van StUF-BG gebaseerd.

De laatst gepubliceerde patch van StUF-BG 2.04 is patch 14 i.v.m. de wens Tijdelijke verblijfsadressen van niet ingezetene te kunnen uitwisselen. 

Zie ook [de discussies m.b.t. StUF-BG 2.04](https://github.com/VNG-Realisatie/StUF-Standaarden/labels/StUF-BG%202.04).
